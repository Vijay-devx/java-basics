# If-Else Practice (Java)

Java programs to practice conditional statements and logical decision-making.

---

## What You Will Learn

- if, if-else, else-if, nested if
- Comparison operators (==, !=, >, <, >=, <=)
- Logical operators (&&, ||)
- Ternary operator
- Boolean conditions
