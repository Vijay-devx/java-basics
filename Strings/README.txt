# Strings Practice (Java)

Java programs to practice string manipulation and common string-based problems.

---

## What You Will Learn

- Creating and reading strings
- Traversing characters in a string
- Comparing strings
- Reversing strings
- Palindrome checking
- Counting vowels, words, and digits
- Character replacement
- Case conversion
- Searching inside strings

---
