# Arrays Practice (Java)

Java programs to practice array creation, traversal, and common array operations.

---

## What You Will Learn

- Creating arrays
- Taking array input
- Traversing arrays using loops
- Searching in arrays
- Reversing arrays
- Finding max, min, sum, average, and second largest

---

## Programs Included

- UserInput.java  
- PrintArray.java  
- CopyArray.java  
- SumOfArray.java  
- AverageOfArray.java  
- MaxOfArray.java  
- MinOfArray.java  
- SecondLargest.java  
- LinearSearch.java  
- CountEven.java  
- ReverseArray.java  
- ReverseUsingTwoPointerSwapping.java  

---

## How to Run

javac FileName.java  
java FileName
